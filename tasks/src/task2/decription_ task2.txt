Создать класс склад, в котором есть стеллажи. Каждый стеллаж может хранить определенный товар. Товар имеет два поля: имя и цена. У каждого стеллажа есть один товар и количество этого товара. Для класса склад создать методы:
 1 Метод, который возвращает стоимость всех товаров на складе
 2 Метод, который возвращает общую стоимость одного продукта
 3 Метод, который возвращается количество всех товаров
 4 Метод, который возвращает цену самого дорогого товара
 5 Метод, который возвращает самую маленькую стоимость товара
 6 Метод, для добавления новых товаров на склад

Считать, что стоимость = цена * количество.
Для множества объектов использовать массивы(коллекции не использовать). 
Для каждого класса переопределять метод toString(), который будет возвращать актуальные данные в строковом представлении. 
Создать класс склад и продемонстрировать работу всех методов, выводом результата на консоль, в отдельном классе Main.
